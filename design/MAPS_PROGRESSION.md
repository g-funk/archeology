# Maps Progression

This documents describers what is found from maps. It can be used to provide input map configs

## Tutorial Collections

There are two collections that can be considered tutorial collections:

### First collection

Contains one shelf, which contains three items. This collection should be completed within first three maps.

### Second collection

Contains two shelves. First shelf contains two normal items, and the second shelf contains one 3-part partial item.

## Tutorial maps

During tutorial maps the player is quickly introduced a few main concepts of the game: items, collections and shelves, partial items, scrap tiems, stamina and recharge.

### Stamina Recharge

During excavation tutorial, recharging stamina is free

### Map 1

Contains a single collectible ITEM that is relatively easy to find and dig out. This unlocks the first collection. The first collection has a single shelf with two items.

Depth: shallow

### Map 2

Contains one SCRAP ITEM and one collectible ITEM. The collectible item goes to the second slot on the unlocked SHELF. The scrap item should be close to player starting point, and the actual item a bit further.

Depth: shallow

### Map 3

Contains the last item for the unlocked SHELF, and one fragment for a partial item in the second tutorial collection. Two scrap items.

Depth: more

### Map 4

Contains one more part for the partial item, two scrap items

Depth: more
